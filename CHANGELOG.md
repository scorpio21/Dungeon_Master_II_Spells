# Changelog

## [1.0.2] - 2025-11-07

### Mejorado

- Sistema de instalación mejorado con Inno Setup
- Workflow de GitHub Actions para builds automáticos
- Optimización del tamaño de los archivos generados
- Mejor manejo de rutas de imágenes

### Corregido

- Problemas de carga de imágenes en el instalador
- Configuración de rutas relativas para mayor portabilidad

## [1.0.0] - 2025-11-06

### Características iniciales

- Primera versión pública
- Proyecto WinForms .NET 8 con formulario editable en Visual Studio
- Cálculo de coste por nivel (Power + símbolos con factor floor(PL1 × (nivel+1)/2))
- Selector de poder, clase y hechizo
- Iconos centrados y en cuadrícula con etiquetas
- Carga de imágenes desde `img/` y `img/posiones/`
- Tooltips con descripción y familia del símbolo
- Documentación con README y licencia MIT
